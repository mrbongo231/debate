# **App Name**: Eloquent Engine

## Core Features:

- Speech Outline Generation: Generates a structured speech outline, with an introduction, several main points, and a conclusion based on the user's topic or quote. This acts as a tool to help the user speak fluently.
- Customizable Practice Timer: Allows users to set a practice timer for speeches with customizable start/stop controls.
- Save Prompts and Outlines: Enables users to save their prompts and generated speech outlines for later review.
- Review Past Prompts: Allows users to access their past prompts and outlines.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) for a professional and trustworthy feel.
- Background color: Dark navy (#1A2A3A) for a sophisticated and modern aesthetic.
- Accent color: Light blue (#79A8D3) to highlight important elements.
- Body and headline font: 'Inter', a sans-serif font for a clear, neutral and modern look.
- Use simple and elegant icons that clearly represent different speech components and settings.
- A clean, intuitive layout with a focus on readability and ease of use. Use a generous amount of whitespace to prevent clutter.
- Subtle animations and transitions to provide feedback and guide the user through the speech preparation process.
